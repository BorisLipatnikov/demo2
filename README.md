основные файлы проекта понижающий преобразователь напряжения с регулируемым выходом на базе pic16f1777 для управления питанием коллекторного мотор-редуктора.
диапазон регулировки напряжения 20 В ... 4 В , входное 24В постоянного.
уровень напряжения задаётся цифровым управляющим сигналом DMX или внутренней логикой в зависимости от режима работы прибора.
настройки адреса и профиля работы DMX задаются удаленно с консоли по RDM или через меню пользователя.
прибор поддерживает режимы работы : удаленный, автономный и прямое ручное управление. 
встроена функция автоотключения при обнаружении блокировки мотора.
